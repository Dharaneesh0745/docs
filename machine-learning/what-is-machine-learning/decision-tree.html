🌳 Machine Learning Approaches

A simple flow-chart to determine which AI model type to use.

  • How is the AI learning?
    Dataset vs. Environment
    • 📊 From a Dataset
      Learning from historical records
      • Do you have labels?
        Is there a target column?
        • ✅ Yes (Supervised)
          Predicting known answers
          • 📈 Regression
            Target is a Number
          • 🏷️ Classification
            Target is a Category
        • ❌ No (Unsupervised)
          Finding hidden patterns
          • 👥 Clustering
            Group similar data
          • 📉 Dimensionality
            Reduce features
    • 🎮 From Environment
      Learning via actions & feedback
      • 🤖 Reinforcement
        Maximize rewards (Trial & Error)
