🤖 What is Machine Learning?

A visual overview of how machines learn — and the three core types of ML.

  • How does the AI learn?
    Dataset vs. Environment
    • 📊 From a Dataset
      Learning from historical records
      • Are outputs labeled?
        Is there a target column?
        • ✅ Yes → Supervised
          Learns from labeled examples
          • 📈 Regression
            Predict a number
            e.g. house price
          • 🏷️ Classification
            Predict a category
            e.g. spam / not spam
        • ❌ No → Unsupervised
          Finds hidden patterns
          • 👥 Clustering
            Group similar data
            e.g. customer segments
          • 📉 Dim. Reduction
            Compress features
            e.g. PCA
    • 🎮 From Environment
      Learning via actions & rewards
      • 🤖 Reinforcement
        Maximize reward via trial & error
        e.g. game-playing AI
